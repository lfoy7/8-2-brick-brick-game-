# 8-2-brick-brick-game-
This project is a brick-destroying game with simple graphics. The game does not have a full objective or scoring system, but it allows the player to create bricks by pressing a button and launch moving balls that interact with them. When the balls collide with the bricks, the bricks are either marked as hit or destroyed depending on their type.
Controls are  as follows: spacebar to make the cirlces and "B" to make the bricks.
